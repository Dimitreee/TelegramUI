## Installation

```
yarn add @xelene/tgui
```

## Development

Follow these steps to start development

### Import styles

```
import '@xelene/tgui/dist/styles.css';
```

### Wrap your app with `AppRoot`

```jsx
import { AppRoot } from '@xelene/tgui';

ReactDOM.render(
  <AppRoot>
    <App />
  </AppRoot>,
  document.getElementById('root')
);
```
